琅嬛

雅性内融，惟道是杖；
天骨疏朗，其馨若兰。

这幅题刻位于扬州逸圃的琅嬛小筑，匾额为金文大篆，楹联为行楷，是典型的“以联解额、以额点题”的园林题咏。匾额二字从右至左为琅嬛（lán huán），非“環靄”。以下严格按照「原文→字词注释→现代文翻译→深度赏析」的结构，为你完整解析：

一、 核心原文

位置 文字内容 说明
匾额 琅嬛（篆体） 传说中的天帝藏书处，亦称“琅嬛福地”
上联（右柱） 雅性内融，惟道是杖 落款：成親王書
下联（左柱） 天骨疎朗，其馨若蘭 上联末字“杖”为仄声，下联末字“蘭”为平声，符合“仄起平收”格律

二、 字词注释

  1. 琅嬛（láng huán）：神话中神仙的洞府，后泛指珍藏书籍的地方，寓意此处为藏书读书的仙境。

  2. 雅性内融：高雅的品性向内融会贯通，指修养内化于心，而非流于表面。

  3. 惟道是杖：宾语前置，即“惟杖道”。杖，以……为拐杖，引申为遵循、依凭。意为只遵循圣贤之道。

  4. 天骨疎朗：天骨，指天生的气质、风骨；疎朗（同“疏朗”），形容性格或气质开朗、豁达，不局促。

  5. 其馨若蘭：馨，香气；典出《孔子家语》“与善人居，如入芝兰之室，久而不闻其香，即与之化矣”，比喻品德高尚，美名远扬。

三、 精确现代文翻译

1. 匾额翻译

琅嬛：神仙藏书的福地。（此处意指：这是一个藏满珍籍、宛如仙境的书斋。）

2. 楹联翻译

  • 上联：高雅的性情早已内化于心中，为人处世只遵循和依靠圣贤的大道。

  • 下联：天生的风骨疏阔明朗，超凡脱俗，其品德的芬芳如同兰花一样清馨。

四、 深度赏析

这组题刻是逸圃精神内涵的最高升华，匾额与对联互为表里，且书法出处显赫，具有极高的艺术价值。

1. 匾额与空间的呼应：从“藏书”到“藏道”

  • 命名寓意：园主将此处命名为“琅嬛”，呼应了你之前看到的“家声传邺架”。如果说“邺架”是强调李氏家族的藏书传统，那么“琅嬛”则是将这个书房的地位提升到了仙境的高度。它不再只是一个物理上的房间，而是园主精神上的避难所和修道场。

2. 对联的哲学内涵：内圣外王的文人理想

这副对联对仗工整，分别阐述了文人修养的两个层面：

  • 上联修“心”（内圣）：“雅性内融”讲的是内化，“惟道是杖”讲的是准则。强调内心的修养必须有大道作为支撑，不能是空洞的清高。

  • 下联修“形”（外王）：“天骨疎朗”讲的是气质，是内心修养外化后的精神面貌；“其馨若蘭”讲的是影响，是品德对他人的熏陶。

3. 书法艺术：皇家风范与金石气

  • 成亲王永瑆：乾隆皇帝第十一子，清代著名书法家。其书法笔力遒劲，结体匀称，与对联中“疏朗、雅性”的气质完美契合。

  • 匾额：采用金文大篆，笔画盘曲，古朴厚重，与“琅嬛”这一古老的神话词汇相得益彰，营造出一种深邃、神秘的历史感。

4. 与前序题刻的闭环

结合之前解析的“心鏡常磨”和“世泽数平渊”：

  • 心鏡常磨是方法（如何修身）；

  • 雅性内融，惟道是杖是目标（修养成什么样的品性）；

  • 琅嬛是环境（在什么样的地方修身）。

这三者共同构成了逸圃作为“儒商园林”的完整逻辑：在仙境般的书房里，通过时时自省，最终达到品德如兰、依道而行的境界。

灯窗作字水沄沄 花底振衣清仩鹳

这是扬州逸圃的另一副经典篆书楹联，书体为吴让之篆书（落款可见“让之吴熙载”），文字采用金文大篆写法，字形繁复古雅。结合篆书字形、平仄格律及园林语境，现将对联辨正、翻译并赏析如下，严格遵循你要求的「精确翻译+深度赏析」结构。

一、 对联原文（辨正与断句）

注：篆书书写为竖排，需从右向左读；上联末字“沄”为平声，下联末字“鹳”为仄声，符合扬州园林楹联“上平下仄”的特殊习惯。

  • 上联（右柱）：镫窗作字水沄沄

  • 下联（左柱）：花底振衣清仩鹳

二、 字词注释

  1. 镫窗：“镫”通“灯”。指灯下窗前，代指书斋苦读或写字的场景。

  2. 沄沄（yún yún）：水流汹涌回旋的样子。此处形容书写时笔墨挥洒流畅，如水流不断；也暗喻文思泉涌。

  3. 振衣：抖落衣上的尘土或花瓣。典出《楚辞·渔父》“新沐者必弹冠，新浴者必振衣”，指整肃衣冠，洁身自好。

  4. 仩（chán）鹳：“仩”通“亶”，此处借作“鹳”的修饰词；“鹳”指鹳鸟，一种栖息于水边的高古之鸟。“清仩鹳”即“清鹳”，指姿态清高、鸣声清亮的鹳鸟。

三、 精确现代文翻译

  • 上联：在灯下窗前挥毫写字，笔墨流淌的姿态如同回旋的水波一般，连绵不绝，充满韵律。

  • 下联：在花丛之下抖落衣衫，整肃仪容，那清高脱俗的气质，仿佛如同水边伫立的鹳鸟一样，洁净而孤傲。

四、 深度赏析

这副对联是吴让之篆书艺术与园林文学的完美结合，与前几副相比，更具动态美与人格美，妙处有三：

1. 一文一武，一动一静（意境之美）

对联描绘了文人两种不同的生活状态，构成了园林生活的完整画卷：

  • 上联写“文事”：聚焦于“作字”，以“水沄沄”比喻笔势的流畅与文思的充沛，是室内的、静态的精神活动，但通过比喻赋予其水流的动态感。

  • 下联写“雅趣”：聚焦于“振衣”，以“清仩鹳”比喻人格的清高与洁净，是室外的、动态的身体活动，但“鹳”的形象又带有静穆、孤高的气质。

2. 篆书笔法与文字内容的高度统一（书法之美）

此联由晚清篆书大家吴熙载（号让之） 书写，其篆书特点是“圆劲流美，舒展飘逸”：

  • 呼应上联：笔画的婉转曲折，恰好呼应了“水沄沄”的水流之态，观其字，如见水波。

  • 呼应下联：字形的修长挺拔，如同鹳鸟的长腿与长颈，也暗合“振衣”时衣袂飘飘的修长感。

  • 金石气：吴让之的篆书源于汉碑额，虽圆转却不软媚，体现了“清仩鹳”那种清高而不柔弱的风骨。

3. 典故的化用与升华（人格之美）

下联“振衣”一词，并非简单的抖衣服，而是化用了屈原的典故，表达了园主人洁身自好、不与世俗同流合污的高尚情操。

  • 园主李鹤生作为清末民初的富商，身处繁华喧嚣的扬州，却能在园林中“花底振衣”，以“清鹳”自比，表明自己虽处商界，内心却向往文人的清高与淡泊。

  • 上联“镫窗作字”则表明，这种清高并非天生，而是通过读书、写字等文化修养得来的。

五、 与前联的关联

这副对联与“窗竹鸣琴韵入弦”联，同写文人雅事，但侧重点不同：

  • “窗竹”联侧重感官的交融（竹声入琴，花香入字），是唯美的；

  • “镫窗”联侧重人格的修养（笔势如水，气质如鹳），是励志的。

两联互补，共同构成了逸圃主人“以文修身，以雅养性”的精神追求。

瓶花落砚香归字 窗竹鸣琴韵入弦

这是扬州逸圃书斋内的一副七言集景联，落款为“己丑春仲 顾大风”，书法为行楷，笔意潇洒。结合对联格律（上联仄收、下联平收）及你提供的图片位置，左柱为上联，右柱为下联。

以下严格按照你要求的格式，进行精确辨字、直译、深度赏析，重点补充此前未提及的格律细节与“通感”修辞的逻辑拆解。

一、 对联原文（校正版）

  • 上联（左）：瓶花落砚香归字

  • 下联（右）：窗竹鸣琴韵入弦

二、 字词注释

  1. 瓶花：书斋中插在瓷瓶里的鲜花，文人书斋“清供”之一。

  2. 落砚：花瓣凋落，恰好落在砚台之中或砚台旁。

  3. 鸣琴：指风吹竹叶发出的沙沙声，酷似琴弦振动的声响，非指真的在弹琴。

  4. 归 / 入：均为动词，意为融入、渗入，是对联的“诗眼”。

三、 精确现代文翻译

  • 上联：瓶中的花瓣飘落在砚台之上，花的清香与墨汁的浓香交融，仿佛最终都凝聚在了书写的文字里。

  • 下联：窗外的翠竹在风中摇曳作响，那声音如同琴声般悠扬，清雅的韵律仿佛直接飘入了琴弦之中。

四、 深度赏析

这副对联是中国园林楹联中“通感”运用的教科书级范例，也是逸圃中最具“书卷气”的一副。它妙在将无形的感官体验转化为有形的艺术创作，构建了一个“物我两忘”的文人境界。

1. 修辞核心：极致的“通感”与“虚实相生”

对联上下两联分别运用了通感，且形成了完美的对仗：

  • 上联（嗅觉 → 视觉/触觉）：

    “香”是嗅觉，“字”是视觉。花香本是流动的，园主却用一个**“归”**字，将无形的香气定格在有形的墨字中。这是一种心理感知——因为爱书爱花，便觉得写出的字都沾染了花的灵气。实景是“瓶花落砚”，虚景是“香归字”。

  • 下联（听觉 → 听觉/触觉）：

    “竹声”是自然之声，“琴韵”是人为之声。风摇竹影本无韵，因听者心中有琴，便觉竹声有了韵律。一个**“入**”字，将墙外的竹声与室内的琴弦（或心中的琴）无缝连接。实景是“窗竹鸣”，虚景是“韵入弦”。

2. 动词炼字：“归”与“入”的哲学意味

  • 归：有“归宿、沉淀”之意。寓意文人的修养，如同花香入字，是日积月累、沉淀下来的结果。

  • 入：有“渗透、融合”之意。寓意人与自然的和谐，不是人去欣赏自然，而是自然主动融入人的艺术生命中。

3. 意境营造：静中取动，小中见大

  • 场景极小：仅仅是书斋一隅的“瓶花”与“窗竹”。

  • 意境极大：上联写文（笔墨纸砚），下联写艺（琴棋书画）。花香入字，是“文气”；竹声入弦，是“雅气”。

  • 动静结合：花落是动，字成是静；竹鸣是动，琴韵是静。一动一静之间，尽显文人“静观万物”的闲适心态。

4. 与“琅嬛福地”的呼应

这副对联位于逸圃的“琅嬛小筑”附近，与匾额“琅嬛”（神仙藏书处）形成呼应：

  • 神仙的境界不仅在于藏书多，更在于人与环境的高度和谐。

  • 在这里，花助文思，竹佐琴兴，人在其中，自然也就成了“神仙”。

世泽数平渊，草木争荣，茵作地 家声传邺架，子孙有幸，好藏书

一、对联原文（辨正+格律定序）

按传统楹联“仄起平收”规则，“地”为仄声、“书”为平声，结合你拍摄的柱位，校正并断句如下：

  • 上联（右柱）：世泽数平渊，草木争荣，茵作地

  • 下联（左柱）：家声传邺架，子孙有幸，好藏书

款识：上联小字“逸圃旧题”，下联为“戊子新正邗上……李桂林书”，是扬州逸圃李氏家族的家训联。

二、核心字词注释（聚焦易混字，极简释义）

  1. 数（shǔ，动词）：凝望、细数，非数量词“几”。

  2. 世泽：祖先积累的恩泽、德业。

  3. 茵：草褥，此处作动词，指绿草像褥子一样铺满地面。

  4. 邺架：典出唐·李泌（邺县侯），其家藏三万卷书，后代指藏书世家的传统，也是李姓家族的文化符号。

  5. 好（hào，动词）：喜爱、爱好，非“好坏”之“hǎo”。

三、精确现代文翻译（逐句对应，直白易懂）

  • 上联：祖先留下的恩泽，能从这平静的池水间凝望体味，园中的草木竞相繁盛，绿草如茵铺满了地面。

  • 下联：家族的声望，传承着李泌藏书世家的风雅传统，子孙们何其幸运，能以喜爱收藏书籍为乐。

四、深度赏析（贴合园林语境，兼顾修辞与家风）

这副对联是扬州逸圃最具家族烙印的家训联，以“园景”写“家风”，是儒商园林“以文传家”的核心体现，妙处有三：

1. 虚实对仗：以园景喻家风，情景交融

上联实写园林，虚喻家道：“平渊”“草木争荣”“茵作地”是逸圃实景，以池水的澄澈、草木的繁茂，比喻李氏家族世泽深厚、子孙兴旺；下联实点家风，虚承典故：“邺架”“好藏书”是家族传统，以李泌藏书的典故，点明家族“重文轻商”的核心价值观。一景一德，上联是“家道之基”，下联是“家道之魂”，对仗工整且逻辑闭环。

2. 炼字精妙：两个动词，道尽文人修养

  • 上联的“数”：将“世泽”这一抽象概念，转化为“凝望平渊”的具体动作，体现了后人对祖恩的敬畏与体悟，而非空洞的感念。

  • 下联的“好”：是全联的“诗眼”。园主强调“好藏书”而非“多藏书”，意为藏书的价值不在数量，而在子孙对书籍的热爱、对学问的追求，这才是家族“有幸”的真正内涵——传承的是志趣，而非财富。

3. 姓氏双关：邺架典故，彰显家族骄傲

园主李鹤生是清末扬州钱庄巨贾，却不以富贵为荣，反而以“邺架家声”自居。“邺架”的典故专属李姓，既呼应了家族姓氏，又将商业资本与文化传承结合，诠释了扬州儒商“贾而好儒，诗书传家”的特质。这也是逸圃所有楹联中，唯一直接关联家族姓氏与传承的一副，是名副其实的“李氏家训碑”。

4. 与逸圃其他楹联的关联

这副联与“心镜常磨”“瓶花落砚香归字”形成呼应：“好藏书”是修身的途径，“心镜常磨”是修身的方法，“瓶花落砚”是修身的意境，三者共同构成了逸圃“读书、修心、养性”的完整精神链条。

與天同壽福仙府 長生不老道人家

这处题刻位于扬州东关街的武当行宫（真武庙），属于道教宫观的山门楹联。建筑门头上方的石额为“武当行宫”，两侧为楷书描金对联。

按照传统楹联“仄起平收”规则（上联末字“府”为仄声，下联末字“家”为平声），结合柱位与道教语境，为你进行完整解析：

一、 核心原文

位置 文字内容 备注

石额（门头） 武当行宫 建筑名称，非传统横批

上联（右柱） 與天同壽福仙府 壽，繁体同“寿”

下联（左柱） 長生不老道人家 長，繁体同“长”

二、 核心字词注释

  1. 與天同壽：道教常用语，指功德或寿命与天地一样长久，此处专指真武大帝的神性。

  2. 福仙府：充满福气的神仙府邸，指代武当行宫。

  3. 長生不老：道教的核心追求，指通过修道达到生命永恒的境界。

  4. 道人家：修道之人的居所，也暗指居住在此地或信奉道教的人能得仙道庇佑。

三、 精确现代文翻译

  1. 石额：武当行宫——武当山真武大帝在扬州的行宫。

  2. 上联：（真武大帝）功德与苍天一样长久，这里是充满福泽的神仙府邸。

  3. 下联：修行者追求长生不老的境界，这里是道法高深的修行人家。

二、 核心释义：“碧润”是什么意思？

直译：碧绿而温润。

这是一个极具画面感的词，专门用来形容被雨水或晨露滋润过的翠竹。

  • 碧：紧扣个园的主题——竹。竹叶的颜色是青碧色，“碧”字点明了眼前的植物景观。

  • 润：呼应个园“宜雨轩”的赏雨文化。竹子喜雨，雨后的竹叶色泽鲜亮、饱含水分，这种状态即为“润”。

匾额取名“碧润”，意为：此处翠竹繁茂，雨露滋润，满目青碧，是一处清幽湿润的雅致空间。

三、 文化由来与典故

“碧润”一词并非生造，它源自中国古典文学，是文人墨客形容竹、水的常用雅词，有两处核心文化源头：

  1. 源自唐诗，形容竹的生机

唐代诗人常以“碧润”描绘竹景。如唐代诗人李群玉《题竹》中有类似意境：“一顷含秋绿，森风十万竿……碧润沾云湿，青光入画寒。”

个园取此二字，既描绘了实景，又赋予了竹林浓厚的唐诗意境。

  1. 源自书法，形容笔墨的气韵

在书法理论中，“碧润”常用来形容笔墨的温润丰腴、富有光泽，如“笔意碧润，如春雨沐林”。

这块匾额的字体是行楷，笔画圆润流畅，其笔墨本身就呈现出一种“碧润”的质感，与文字内容形成了“字如其意”的美学呼应。

这个八角门是进入个园竹林隧道的入口，匾额的设置有两层深意：

门外是开阔的庭院，门内是幽深的竹林小径。“碧润”二字提前给游人一种心理暗示：穿过这扇门，你将进入一个绿意盎然、湿润清凉的世界，瞬间从喧嚣转入清幽。

个园以“四季假山”闻名，而这片竹林是夏山的重要组成部分。夏季多雨，竹子繁茂，“碧润”精准地概括了夏日竹林雨后的神韵，与夏山的“清凉、湿润”主题高度契合。

站在丛书楼前，读此联，仿佛能闻到芸香草的清香，看到竹影映在书案上的斑驳，感受到园主人在繁忙的盐务之余，躲进书楼与竹为伴、与书为友的那份淡泊与宁静。

秋從夏雨声中入；春在寒梅蕊上尋

这是扬州个园·住秋阁的核心楹联，也是个园所有题刻中最具哲学意味、最能体现“四季假山”造园逻辑的一副名联。匾额“住秋阁”由李圣和题写，对联以行书书写，墨色雅致。

这副对联的绝妙之处在于不写秋之萧瑟，而写四季的流转，完美诠释了个园“春去夏来，夏去秋至，秋去春来”的时空设计。以下严格按照你的要求，进行繁简对照、注音、古用法解析、精准翻译及深度赏析：

一、 核心原文（繁简对照+注音+上下联定序）

匾额与对联均遵循从右至左的传统书写顺序。格律为仄起平收：上联末字“入”（仄声），下联末字“尋”（平声）。

二、 核心古用法与造园语境解析

此联文字看似浅显，实则运用了通感与时空蒙太奇的手法，且紧扣个园的实景游览路线：

  1. 住秋（zhù qiū）：文人雅语。秋天易逝，古人常建“住秋亭/阁”，意为试图将美好的秋景留住，表达了对时光的眷恋。

  2. 秋從夏雨声中入：个园实景。住秋阁紧邻夏山（湖石假山）。夏日傍晚，雨声打在夏山的竹林和湖石上，雨过天晴，凉意顿生，秋天仿佛是随着雨声悄悄“潜入”的。

  3. 春在寒梅蕊上尋：个园实景。住秋阁对面即是冬山（宣石假山），冬山上植有腊梅。当秋尽冬来，梅花含苞待放时，人们在寒梅的花蕊中，便能“寻觅”到春天的消息。

  • 匾额·住秋阁：
    一座试图留住秋日美景、让人沉醉于秋韵之中的楼阁。

  • 上联：
    秋天的脚步，是从夏日渐渐稀疏的雨声中，悄然潜入人间的。

  • 下联：
    春天的生机，正藏在寒冬腊月梅花的花蕊里，等待人们去探寻发现。

四、 深度赏析（个园四季的灵魂之联）

这副对联是个园**“四季假山”设计理念的文字版说明书**。它没有直接描写秋山的壮丽，而是写出了季节与季节之间的过渡与连接，妙处有三：

  1. 时空流转：打通“夏、秋、冬、春”四季循环
    这是全联最精彩的地方：

  • 上联（夏→秋）：写听觉。秋不是突然来的，而是在夏雨的尾声中到来的。这对应了个园的游览路线：从夏山走到秋山，中间经过住秋阁，雨声是季节转换的信号。

  • 下联（冬→春）：写视觉。春不是在冬后才来的，而是在冬梅的花蕊中就已孕育。这对应了个园的布局：冬山（寒梅）紧邻春山（竹子），在冬山赏梅，便能望见春山的绿意。

整副对联形成了一个闭环：夏→秋→冬→春，四季轮回，生生不息。

  1. 炼字精妙：“入”与“尋”，极具动感与情趣

  • 入：一个字写出了秋天的轻盈与含蓄。秋不是轰轰烈烈地“来”，而是悄悄地“入”，如同一位不速之客，随着雨声钻进窗棂。

  • 尋：一个字写出了春天的珍贵与希望。春藏在寒梅蕊中，需要人用心去“寻”。这既是写探梅之趣，也暗喻了文人对美好未来的追求。

  1. 造园哲学：“咫尺山林，多方胜景”

个园的面积不大，却要表现四季，必须讲究**“邻借”**。

  • 住秋阁作为秋山的观赏点，一抬头能看到夏山的雨，一低头能看到冬山的梅。

  • 这副对联告诉游人：站在住秋阁，你看到的不仅是秋，而是夏末、秋初、冬至、春信。这就是中国园林“以小见大”的最高境界。

五、 与前文脉的呼应

这副对联与你之前看到的“園開四季花”形成了具体与抽象的呼应：

  • “園開四季花”是总写，概括了个园有四季之景；

  • “秋從夏雨声中入，春在寒梅蕊上尋”是特写，具体描绘了四季是如何在空间上交错、在时间上流转的。

总结

这是一副充满禅意与诗意的对联。

它不仅是在写景，更是在写时间的哲学：季节的更替不是断裂的，而是连续的、相互包容的。正如人生，寒冬之中孕育着春机，盛夏之后便是清秋。

站在住秋阁，读此联，你会发现：个园的秋，是有声音的（夏雨），也是有希望的（寒梅）。

筑室构大夏，当月瞻望，喜见梓材能作室；棣栭松柏，梧槚桐梓，猗猗新篁

这是扬州个园中最具“金石气”的一副楹联，位于清颂堂（个园主人的正厅）。（不是个园 ，是汪氏小苑）

这副对联由近代扬州著名学者、书法家**陈延韡（wěi）在戊午年（1918年）撰文并以大篆（金文）**书写。其特殊之处在于：文字是上古金文，内容是集句古文，书法是金石篆书，且上下联字数不等（长联对短联），属于文人雅集的“集句联”。

二、 核心古汉字与典故解析（读懂此联的门槛）

此联大量使用**《诗经》与《尚书》中的词汇，且多为树木与建筑术语**，寓意极深：

  1. 梓材（zǐ cái）：语出《尚书·梓材》。原指梓木之材，后喻指培养人才。在园林语境中，双关指“建造房屋的良材”和“家族中的贤才”。

  2. 大夏（dà xià）：即“大厦”。指高大的房屋，此处特指清颂堂。

  3. 棣、栭、梧、槚、桐、梓：均为古代优质建筑木材或嘉木。

  • 槚（jiǎ）：古汉字，指楸树或茶树，此处指良木。

  • 栭（ér）：屋檐下的柱子，此处代指木材。

  1. 猗猗（yī yī）：语出《诗经·淇奥》“绿竹猗猗”。形容竹子美盛、婀娜的样子。

  2. 新篁（xīn huáng）：指新生的竹子。“篁”是竹的通称，新篁即春笋长成的新竹，是个园的标志。

三、 精确现代文翻译

这副对联运用了赋的手法，上联言志，下联铺陈。

  • 上联译文：
    建造这所宏大的宅第（清颂堂），当月色照耀时登高远望，欣喜地看到（家族中）有像梓木一样的良材，能够继承家业、建造美好的家园。

  • 下联译文：
    这里汇聚了棣、栭、松、柏、梧、槚、桐、梓等各种珍贵的木材；更有那风姿绰约、生机勃勃的新生翠竹。

四、 深度赏析（清颂堂的“家族宣言”）

这副对联挂在个园最庄重的正厅“清颂堂”，其意义远超普通写景，是盐商家族的自我期许与文化标榜。

  1. 内容双关：以“材”喻“才”
    这是此联的核心。下联罗列了大量的树木（松、柏、桐、梓、竹），表面是说建造房子用的是良材，实则是比喻家族子弟人才济济。
  • “喜见梓材能作室”：借用《尚书》典故，表达了园主人希望子孙后代皆为栋梁之材，能够光宗耀祖，把家族的基业（大厦）建设得更好。
  1. 书法与空间的契合：金石气配正厅

  • 书体：使用大篆（金文），笔画盘曲、苍劲古朴，具有浓厚的“金石气”。

  • 效果：清颂堂是个园的礼仪中心，用于接待贵宾、举行家宴。篆书的庄重、典雅，恰好匹配正厅的威严与正统，与后花园“宜雨轩”“住秋阁”的行书、隶书（飘逸、灵动）形成鲜明对比。

  1. 点题个园：竹的升华
    下联最后以“猗猗新篁”收尾，巧妙地回扣了个园的“竹”主题。

  • 从各种珍贵的硬木（代表财富与坚实），过渡到清雅的新竹（代表气节与新生）。

  • 寓意家族不仅富贵（有栋梁之材），而且有文化、有气节（有猗猗新篁），完美诠释了扬州盐商“贾而好儒”的最高境界。

五、 总结

这副篆书对联是个园所有楹联中文化门槛最高的一副。

它不写风花雪月，不写山水之乐，而是直抒胸臆。上联讲“筑室育人”，下联讲“材木茂盛”。站在清颂堂前，你看到的不仅是古老的文字，更是百年前扬州盐商家族希望子孙成才、家业长青的殷切期望。

这副楹联位于扬州何园（寄啸山庄）的船厅，是何园最经典的“镇园之联”，由晚清扬州书法家李钟豫（字毓如）题写。对联以拟人化手法，将船厅“无水却似有水、非船却宛如船”的造景精髓写绝，下联更是直接集宋代陆游名句而成。

  1. 集句典故（关键）

下联“花为四壁船为家”直接引自南宋陆游《同何元立赏荷花追怀镜湖旧游》：“三更画船穿藕花，花为四壁船为家。”李钟豫巧妙截取此句，与自创的上联对仗，既有宋诗的豪放，又添园林的雅致。

船为家：何园船厅是一座旱船（不系舟），四周地面以鹅卵石、瓦片铺成水波纹，形似船行水上。“船为家”既指船厅的建筑形态，也暗合园主人何芷舠辞官归隐后，以园为家、寄情山水的心境。

“月作主人”“梅作客”：将月亮、梅花拟人化，打破人与物的界限，营造出“物我两忘、主客尽欢”的诗意氛围。

  • 上联：皎洁的明月仿佛成了这庭院的主人，而傲立的梅花则像是受邀而来的贵客。

四、 深度赏析（船厅意境+艺术手法+文脉内涵）

这副对联是建筑、文字、意境三者高度统一的典范，专为船厅量身定制，妙处有三：

  1. 虚实相生，完美契合船厅造景

  • 虚写（水）：下联虽未提“水”，但“船为家”三字，结合地面的水波纹，让人仿佛置身于烟波浩渺之中，以“船”衬“水”，虚实相生。

  • 实写（景）：上联写“月”“梅”，下联写“花”“船”，均是船厅周围的实景。船厅旁植有梅花、牡丹，花开时节，确如“花为四壁”。

  1. 主客互易，尽显文人雅趣

  • 月作主人：通常人是主人，月是客体。此处反客为主，写出了文人对月亮的偏爱——在清幽的夜晚，月亮主宰着庭院的意境，人则成了享受这份宁静的过客。

  • 梅作客：梅花傲霜斗雪，是文人高洁品格的象征。以梅为客，既赞美了梅花的品格，也暗示了园主人自身的高雅情趣，可谓“以梅自况”。

  1. 典故活用，寄托归隐情怀
    园主人何芷舠曾任江汉关监督，一生与船、水打交道。辞官后建造船厅，悬挂“花为四壁船为家”的对联，既是对自己过往生涯的追忆，也是对陶渊明“归去来兮”归隐生活的向往。将官船变为“家园”，将尘世喧嚣隔绝在外，尽显“寄傲舒啸”的文人风骨。

五、 与个园楹联的对比（补充阅读）

你此前赏析的个园楹联，多以**“四季假山、竹石之趣”为核心，文字典雅、格律严谨；而这副何园船厅联，更具宋诗的流畅自然与浪漫色彩**。个园重“理”（四季秩序），何园重“情”（归隐之乐），这也是两大名园文风的细微差异。

总结

这是一副**“借景抒情、以典言志”的佳作。上联以拟人手法写月夜梅景的雅致，下联集陆游名句写船厅归隐的闲适。全联仅14字，却将何园船厅的建筑之奇、花木之盛、意境之幽、主人之志**全部概括，读来朗朗上口，余味无穷。

赏析讲解

  1. 上联意境
    此句化用了王维《汉江临泛》中“山色有无中”的名句。它描绘了站在平山堂远眺江南时，山峦在烟岚雾气里若隐若现的朦胧美。“长在”二字赋予了山光永恒的意味，暗含着自然之美不因人事变迁而改变的哲理。

  2. 下联哲思
    下联从写景转向抒情议论。它以反问的语气，提出“诗意”这种超越时空的审美体验，在古人和今人心中是共通的。这巧妙呼应了平山堂作为欧阳修、苏轼等文人雅集之地的历史，点明了此处的人文精神一脉相承。

  3. 整体妙处

曉起憑欄六代青山都到眼；晚來對酒二分明月正當頭

对联原文

上联：晓起凭栏六代青山都到眼

下联：晚来对酒二分明月正当头

现代文翻译

  • 上联：清晨起身，凭靠着栏杆眺望，曾经见证六朝兴衰的青山，全都映入了眼帘。

  • 原文：晓起凭栏六代青山都到眼

  • 现代汉字：晓起凭栏六代青山都到眼（“六代”指东吴、东晋、宋、齐、梁、陈，是建都于南京的六个朝代，代指江南的历史沧桑；“凭栏”为古用法，意为“倚靠栏杆”）

  • 下联：傍晚归来，对月饮酒，那扬州城里最富诗意的“二分明月”，正高悬在头顶。

  • 原文：晚来对酒二分明月正当头

  • 现代汉字：晚来对酒二分明月正当头（“二分明月”出自唐诗“天下三分明月夜，二分无赖是扬州”，特指扬州月色的秀美；“对酒”意为“面对着美酒饮酒”）

赏析讲解

  1. 上联：抚今追昔，气象开阔
    上联以“晓起凭栏”开篇，点明了清晨登高望远的场景。“六代青山”不仅是眼前的实景，更承载着厚重的历史感，让人联想到扬州城见证的千年兴衰。“都到眼”三个字，既写出了平山堂地势高旷、视野极佳的特点，也暗含着“以史为鉴，尽收眼底”的豪迈与感慨。

  2. 下联：诗酒趁月，意境优美
    下联则转向夜晚的闲情逸致。“晚来对酒”营造出一种悠然自得的氛围，而“二分明月正当头”则是点睛之笔。它巧妙化用了描写扬州的千古名句，将扬州的月色之美与平山堂的雅集之乐完美结合，让人体会到欧阳修、苏轼等文人在此把酒赏月的诗意与浪漫。

  3. 整体妙处
    这副对联通过“晓”与“晚”的时间对比，“青山”与“明月”的景物对照，将平山堂的地理形胜、历史底蕴与人文雅趣熔于一炉。它不仅是一幅优美的山水长卷，更是一首无声的咏史怀古诗，读来令人心旷神怡，也深刻诠释了此地“文章奥区”的文化魅力。

平山堂 镇堂之宝

对联原文

上联：

幾堆江上畫圖山，繁華自昔。試看奢如大業，令人訕笑，令人悲。

涼應有些逸興，只贏得廿四橋頭，簫聲月色。

下联：

詠切莫把穰苴濁酒，便當住六一翁。況餘韻風流，

一派竹西歌吹路，傳誦於今。必須才似廬陵，方可遨遊，方可獻。

简体翻译

上联：

几堆江上画图山，繁华自昔。试看奢如大业，令人讪笑，令人悲。

凉应有些逸兴，只赢得廿四桥头，箫声月色。

下联：

咏切莫把穰苴（ráng jū）浊酒，便当住六一翁。况余韵风流，

一派竹西歌吹路，传诵于今。必须才似庐陵，方可遨游，方可献。

现代文翻译

上联

眼前江上连绵的青山，宛如一幅天然图画。这里的繁华景象自古如此。

试看那隋炀帝当年奢华无度的“大业”盛世，最终只留下让人耻笑、又让人悲叹的结局。

在这清凉的夜晚，我本应有几分超然的兴致，却只记得“二十四桥”的典故，

耳边仿佛还回荡着当年的箫声，眼前也依旧是那片迷人的月色。

  • 古字/古用法说明：

  • “大业”：指隋炀帝杨广的年号，代指隋朝的奢华与短暂。

  • “廿四桥”：即“二十四桥”，“廿”是“二十”的古写。

  • “逸兴”：指超逸豪放的意兴。

下联

请不要以为，喝着浊酒，就能自比为六一居士欧阳修。

何况他留下的风流余韵，至今还在“竹西佳处”的歌吹路上流传。

必须要有像欧阳修（庐陵人）那样的才华，才能真正在这里遨游山水，

也才配得上为这里献上佳作。

  • 古字/古用法说明：

  • “六一翁”：指欧阳修，他晚年自号“六一居士”。

  • “庐陵”：是欧阳修的故乡（今江西吉安），此处代指欧阳修本人。

  • “竹西”：出自杜牧诗句“谁知竹西路，歌吹是扬州”，代指扬州的繁华胜地。

赏析讲解

  1. 上联：吊古伤今，感慨深沉
    上联从眼前的青山写起，由景入史，借隋炀帝“大业”奢华而亡的典故，抒发了对历史兴亡的感慨。结尾处的“廿四桥头，箫声月色”，又将思绪拉回到扬州的诗意夜景中，悲慨中不失风雅，充满了历史的厚重感。

  2. 下联：致敬前贤，立意高远
    下联则是对欧阳修的致敬与自勉。它以“莫把穰苴浊酒，便当住六一翁”的反诘，强调了欧阳修的才学与地位无人能及。“必须才似庐陵，方可遨游，方可献”一句，既是对前贤的推崇，也暗含了对后世文人的期许，格调高远。

  3. 整体妙处
    这副长联是平山堂所有对联中最具历史纵深感的一副。它将扬州的自然山水、历史兴衰与人文精神完美融合，上联以史为鉴，下联以贤为师，既表达了对历史的反思，也抒发了对欧阳修的景仰。全联文辞典雅，用典精妙，读来荡气回肠，是不可多得的佳作。
